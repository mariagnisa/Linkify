## Description
This is like a reddit clone with a lord of the rings theme. You can share your links, comment on others link and vote on links. You have a profile page with your account settings and you can visit other peoples profile pages.

## How to start

- Git clone https://github.com/mariagnisa/Linkify.git e.g. clone this repo
- Create a database and import the sql-file, which are in th backup folder
- Set up a local server for the repos root, like MAMP.
- Everything is included.
- Browse to your serveradress.
- Enjoy
